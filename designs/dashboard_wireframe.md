# Dashboard Wireframe

## Text-Based Representation of Library Dashboard UI

```
+----------------------------------------------------------+
|                        Library Dashboard                 |
+----------------------------------------------------------+
| Sidebar                         |          Content Grid    |
|----------------------------------------------------------|
| [Home]     [Search Bar ________________]                 |
| [Explore]                                                  |
| [Categories]                                               |
|   - Romance                                                |
|   - Mystery                                                |
|   - Sci-Fi                                                 |
|   - Fantasy                                                |
|   - Non-Fiction                                            |
| [Account]                                                  |
|----------------------------------------------------------|
|                          Book Title 1  [Details]         |
|                          Book Title 2  [Details]         |
|                          Book Title 3  [Details]         |
|                          Book Title 4  [Details]         |
|                          Book Title 5  [Details]         |
+----------------------------------------------------------+

Legend:
[Home, Explore, Categories, Account]: Navigation Links on the Sidebar
[Search Bar]: A utility to search books by title, author, or category
[Book Title]: List of books displayed in a grid with a 'Details' button.
```

### Sections Breakdown

**1. Sidebar:**
- Located on the left side, acting as the primary navigation panel.
- Contains navigation links to:
  - Home
  - Explore
  - Categories (expandable with sub-categories by genre: Romance, Mystery, Sci-Fi, Fantasy, Non-Fiction)
  - Account

**2. Search Bar:**
- Positioned at the top center of the Content Grid.
- A text-input field with a submit button to search for specific books.

**3. Content Grid:**
- Central content area displaying a scrollable list of books.
- Each book listing includes the title and a "Details" button.

**4. Header:**
- The title "Library Dashboard" centered at the top of the page.
